# x402 with Custom Solana Facilitator

Pay-per-request API using x402 with custom facilitator service on Solana devnet.

## Quickstart

From the project root:

```bash
npm install
```

### Step 1: Start the x402 server

```bash
npm run start:server
```

### Step 2: Run the client to test the premium service endpoint

```bash
npm run start:client
```
